# 电商客户价值分析（RFM 模型）

## 项目背景

电商平台每天产生大量交易数据，但如何从数据中识别出**高价值客户**、**潜在流失客户**，是运营决策的关键。本项目使用经典的 **RFM 模型**，对电商公开数据集（Olist）进行分析，将 9.6 万名客户划分为 5 个层级，并为每个层级提供针对性的运营建议。

## 数据集

- **来源**：(https://www.kaggle.com/datasets/ayushshrivastva13/e-commerce-customer-transactions-dataset)  （Kaggle）
- **核心表**：订单表（orders）、订单商品表（order_items）、客户表（customers）、产品表（products）

## 分析方法

### RFM 模型

| 维度 | 定义 | 计算公式 | 业务含义 |
|------|------|----------|----------|
| **R** (Recency) | 客户最近一次购买距今天数 | 分析基准日 - 最后下单日期 | R 越小 → 客户越活跃 |
| **F** (Frequency) | 客户购买次数 | 统计客户订单数 | F 越大 → 客户越忠诚 |
| **M** (Monetary) | 客户总消费金额 | 订单金额求和 | M 越大 → 客户价值越高 |

### 客户分层标准

| R_score | M_score | 客户层级 |
|---------|---------|----------|
| ≥4 | ≥4 | 高价值活跃客户 |
| ≥4 | ≥2 | 潜力客户 |
| ≤2 | ≥4 | 高价值流失风险 |
| ≤2 | ≤2 | 低价值流失客户 |
| 其他 | 其他 | 一般客户 |

## 分析结果

### 客户分布

| 客户层级 | 人数 | 占比 |
|---------|------|------|
| 一般客户 | 34,625 | 35.9% |
| 高价值活跃客户 | 16,013 | 16.6% |
| 低价值流失客户 | 15,889 | 16.5% |
| 潜力客户 | 15,028 | 15.6% |
| 高价值流失风险 | 14,923 | 15.5% |

### 关键发现

1. **高价值流失风险客户占比 15.5%**  
   这批客户历史消费金额高，但已长期未购买（Recency 中位数超过 100 天），是**最容易挽回的价值群体**。

2. **高价值活跃客户仅占 16.6%**  
   这批客户是核心利润来源，需要重点维护，防止流失。

3. **潜力客户占比 15.6%**  
   这批客户近期活跃但客单价较低，通过交叉销售有较大提升空间。

### 高价值客户品类偏好

高价值活跃客户最偏好的 5 个品类：

| 排名 | 品类 | 购买次数 |
|------|------|----------|
| 1 | 美容健康 | 2,075 |
| 2 | 钟表礼品 | 1,875 |
| 3 | 家居布艺 | 1,802 |
| 4 | 家具装饰 | 1,433 |
| 5 | 电脑配件 | 1,373 |

## 业务建议

| 客户层级 | 建议策略 | 预期效果 |
|---------|---------|----------|
| 高价值活跃客户 | VIP 专属权益、会员积分加速 | 提升复购率，降低流失 |
| 高价值流失风险 | 限时高额召回券（满 300 减 60） | 预计召回率 10%-15% |
| 潜力客户 | 交叉销售、满减券提升客单价 | 客单价提升约 20% |
| 一般客户 | 自动化邮件营销 | 低成本触达 |
| 低价值流失客户 | 大促时统一触达 | 控制运营成本 |
